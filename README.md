# CodeVibe Collective ? Story-Driven Portfolio

This project is a single-page, story-first portfolio experience for the fictional creative tech studio **CodeVibe Collective**.

Built with **Next.js 14**, **Tailwind CSS**, **Framer Motion**, and a sprinkle of confetti magic, it delivers a playful yet polished scroll journey that highlights the team, their origin story, and their projects.

## Features

- Hero section with animated gradients, floating emojis, and a cheeky tone
- Timeline-style "Our Story" section with motion-powered storytelling beats
- Interactive team grid with hover reveals and placeholder portraits
- Animated project cards with hover sparkles
- Contact form featuring a light-hearted CTA, social links, and confetti on submit
- Floating background icons and smooth parallax-inspired motion
- Light/dark mode toggle where the sun swaps places with a coffee cup
- Fully responsive layout with Tailwind utility classes

## Getting Started

```bash
# install dependencies
npm install

# run the development server
npm run dev

# build for production
npm run build

# start the production server
npm start
```

Open `http://localhost:3000` in your browser to start vibing.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [canvas-confetti](https://www.npmjs.com/package/canvas-confetti)

## Customizing

- Update section content in `app/page.tsx` and the `components/` directory.
- Tailwind theme overrides live in `tailwind.config.js`.
- Global styles, gradients, and keyframes reside in `app/globals.css`.
